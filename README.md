## <a name="setup"></a>Installation/Setup

### Checkout the Code 
  ```
  $ git clone git@github.com:AnthonyKenny98/Caecilius.git
  $ cd Caecilius
  ```
### Create and Init Virtual Environment
  ```
  $ virtualenv venv
  $ source venv/bin/activate
  ```
### Install Dependencies
  ```
  $ pip3 install -r requirements.txt
  ```
